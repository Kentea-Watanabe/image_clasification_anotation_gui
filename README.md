# image_clasification_anotation_gui

## 概要
Pythonのtkinterで開発したクラス分類器用のGUIアノテーションツール

## 環境
* python 3.6.5

## 使い方
```
git clone https://github.com/Kentea-Watanabe/image_clasification_anotation_gui.git
```

```
python -m venv .venv
```

```
pip install requirements.txt
```
